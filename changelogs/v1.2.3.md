# 更新日志

## [1.2.3] — 2026-06-05

### 新增功能

#### 智能体连通性自动检测
- 新增后台定时检测服务，按可配置的间隔周期性地测试所有已启用智能体的 API 连通性
- 支持单智能体手动测试和「全部检测」一键批量检测

#### 屏蔽词词库升级
- 内置 100+ 条预设屏蔽词，覆盖脏话辱骂、色情低俗、暴力威胁、自残自杀四大类
- 屏蔽词默认启用，首次启动时自动填充

#### 课堂管理增强
- 课堂看板新增模式标签（标准模式 / 分组模式 / 高级模式）
- 分组/高级模式下展示小组真实学生成员列表

#### 数据管理改进
- 报表导出移除 Token 消耗相关列
- 跨设备迁移提示增加 uploads 文件夹路径详情

#### 更新日志
- 关于页面新增更新日志查看功能，支持交互式展开/收起

### 修复

- 防止重复启动多个程序实例
- 屏蔽词违规警告事件未正确推送到教师端的问题
- 学生拉黑/取消拉黑事件未同步到教师端的问题
