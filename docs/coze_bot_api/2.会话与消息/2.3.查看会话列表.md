查看指定智能体的会话列表。 

说明 

- 调用此API之前，应确认当前使用的访问密钥拥有智能体所在工作空间的权限。 

- 仅支持通过此API查看智能体在API或ChatSDK渠道产生的会话。 

- 仅支持查询本人创建的会话。

## 错误码

如果成功调用扣子编程的 API，返回信息中 code 字段为 0。如果状态码为其他值，则表示接口调用失败。此时 msg 字段中包含详细错误信息，你可以参考错误码文档查看对应的解决方法。
