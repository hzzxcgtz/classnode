## 删除会话

删除指定的会话。 

仅会话创建者可以删除会话。 


基础信息

## 错误码

如果成功调用扣子编程的 API，返回信息中 code 字段为 0。如果状态码为其他值，则表示接口调用失败。此时 msg 字段中包含详细错误信息，你可以参考错误码文档查看对应的解决方法。
