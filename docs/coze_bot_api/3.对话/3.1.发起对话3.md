## 非流式响应

在非流式响应中，无论服务端是否处理完毕，立即发送响应消息。其中包括本次对话的chat_id、状态等元数据信息，但不包括模型处理的最终结果。 

非流式响应不需要维持长连接，在场景实现上更简单。但通常需要客户端主动查询对话状态和消息详情才能得到完整的数据。你可以通过接口B查看对话详情确认本次对话处理结束后，再调用B查看对话消息详情接口查看模型回复等完整响应内容。流程如下：
