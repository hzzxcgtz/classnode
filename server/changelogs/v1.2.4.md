# 更新日志

## [1.2.4] — 2026-06-05

### 新增功能

#### 仪表盘重构
- 全面引入 recharts 图表库，使用 BarChart / PieChart 替换原自定义 SVG 环形图
- 顶部新增核心 KPI 行（智能体/班级/学生/进行中课堂/总互动），一目了然
- 四宫格分区展示：AI 智能体（接入方式分布 + 连通性健康状态）、班级管理（最常用班级排行）、课堂管理（状态环形图 + 模式分布条形图）、数据管理（已结束课堂/参与人次/文字量/平均时长）

#### 系统屏蔽词支持逐条删除
- 展开系统屏蔽词分类后，每个词条右侧显示「×」删除按钮
- 删除后分类内该词条立即消失，数量同步更新
- 「恢复预设」仍可一键补回所有已删除的默认屏蔽词

### 修复

- 跨设备迁移提示框样式优化，uploads 路径改为下划线虚线提示，hover 弹出气泡样式更清晰
