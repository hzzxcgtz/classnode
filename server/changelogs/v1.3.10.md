# 更新日志

## [1.3.10] — 2026-06-10

### 新增

#### 智谱清言（清言智能体）平台接入
- 新增「清言智能体」平台支持（智谱清言），接入流式和非流式对话 API
- 支持 Assistant ID + API Key + API Secret 两步鉴权
- 支持图片和文档文件上传识别

#### 多平台完善
- Coze 低代码：图片上传改用正确格式，支持多模态理解
- 文心智能体：新增 threadId 持久化，实现上下文记忆
- 智谱清言：新增 conversation_id 持久化，实现上下文记忆
- Coze 编程：通过 session_id 维持对话上下文

#### 智能体列表优化
- 平台标签更新：Coze → Coze 低代码
- 不支持图片的平台显示「无图片」标签
- 特征标签改用清淡灰色，与平台主标签视觉区分

#### 投屏二维码优化
- 增加扫码提示：建议使用相机扫码，微信/支付宝可能无法识别

### 修复

#### 课堂智能体切换
- 修改课堂智能体时，旧智能体不再残留

#### 历史消息调整
- 本地历史消息从 20 条调至 30 条，支持更长对话记忆

### 文档

- 使用指南更新平台支持描述，覆盖四种接入方式
- README 补充首次启动数据库初始化提示
