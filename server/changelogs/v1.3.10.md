# 更新日志

## [1.3.10] — 2026-06-10

### 新增

#### 智谱清言（清言智能体）平台接入
- 新增「清言智能体」平台支持（智谱清言）
- 完善支持图片和文档文件上传识别

#### 配置帮助截图功能
- 新增配置帮助图片画廊页面，每个平台首个输入框旁显示「配置截图」标签，点击后在浏览器新标签页全屏展示操作步骤截图，支持左右箭头切换和编号显示

#### 全平台配置截图
- Coze 低代码：6 张操作步骤截图（登录 → 创建 → 配置 → 发布 → 令牌 → 填写）
- Coze 编程：3 张操作步骤截图（API URL → Project ID → API Token）
- 文心智能体：3 张操作步骤截图（登录 → 创建 → API 配置）
- 智谱清言：6 张操作步骤截图（登录 → 创建智能体 → 配置 → API Key → Secret → 填写）

#### 智能体列表优化
- 不支持图片的平台显示「无图片」标签
- 不支持流式输出的平台显示「非流式」标签

### 优化

#### 屏蔽词管理优化
- 系统屏蔽词四大类别仅显示分类名称和词条数量，不再展开具体词条内容

#### 使用指南完善
- 二维码扫码引导中的 IP 地址改为「教师机局域网IP」，避免 192.168 网段引起歧义

### 修复

#### 课堂智能体切换
- 修改课堂智能体时，旧智能体不再残留
