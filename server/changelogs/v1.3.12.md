# 更新日志

## [1.3.12] — 2026-06-11

### 新增

#### 学生端教师消息回顾面板（优化）
- 教师消息在页面顶部栏新增「消息」按钮，点击展开下拉面板，按时间排序展示所有历史教师消息

#### 智能体接入引导优化
- 平台选择下方新增「不知道怎么选平台？前往查看各平台特点 →」链接，点击跳转使用指南第二节
- Coze 低代码 / Coze 编程新增温馨提示：免费点数每日 0 点重置，当天至少登录一次即可正常使用
- Coze 编程、文心智能体额外提示：受 API 功能限制，不支持图片理解

#### 截图画廊翻页优化
- 翻页按钮放大，改用 SVG 箭头确保完美居中
- 页码中间新增「点击翻页」提示文字

### 优化

#### 课堂设置锁定（重要）
- 课堂创建后仅可修改名称，AI 智能体、小组分配等配置改为只读展示，不再允许编辑
- 更新使用指南中的相关描述：「课堂创建后仅可修改名称，AI 配置创建时即固定」

#### 智能体接入表单调整
- 智谱清言的 API Key 与 API Secret 交换位置
- 配置截图按钮从标签旁移至所有输入框下方居中显示，样式重新设计

### 修复

- 修复课堂详情页在线人数初始化时序问题
- 修复学生强制关闭浏览器后重新进入时可能出现的登录异常
