# 更新日志

## [1.3.13] — 2026-06-13

### 新增

#### 多网卡 IP 适配
- 控制面板窗口：多网卡时显示 IP 下拉选择，切换教师/学生访问地址
- 投屏码界面：多网卡时显示 IP 下拉选择，QR 码和地址联动更新

#### 重置管理密码
- 控制面板窗口新增「重置管理密码」按钮
- 教师登录页增加提示：遗忘密码可在控制面板中重置

#### 数据列表通用翻页组件
- 新增 `Pagination` 组件：支持首页、上一页、页码/总页、下一页、尾页、指定页跳转、每页条数选择

### 优化

#### 智能体管理优化
- 停用智能体时，卡片左下角状态同步显示「停用」标识
- 连通性测试按钮增加旋转动画 + 流动进度条，提升等待体验

#### 班级管理布局优化
- 右侧班级概览区改为紧凑布局，头部高度缩减约 50%
- 未分配学生姓名框增加 `nowrap` 和等宽 flex，防三字姓名折行

### 修复

- 修复课堂设置弹窗"保存设置"重复按钮
