# 更新日志

## [1.3.6] — 2026-06-07

### 新增

#### 桌面端状态面板窗口
- 新增桌面窗口，实时显示服务运行状态、访问地址、局域网 IP
- 支持窗口内一键启动/停止服务
- 教师端/学生端地址一键复制与打开浏览器
- 系统托盘左键显示面板，右键弹出菜单
- 托盘菜单新增「显示面板」「项目地址」选项
- 关闭窗口最小化到托盘，双击任务栏图标恢复

### 优化

#### 智能体连通性检测优化
- 取消定时自动检测，改为启动时检测一次，节省 API 调用额度
- 学生与 AI 对话时若连接失败，实时通知教师端
- 保留单个智能体手动检测功能

#### 屏蔽词系统增强
- 系统屏蔽词和自定义屏蔽词支持一键启用/禁用
- 禁用后的屏蔽词以半透明显示，不再参与过滤

#### 弹窗提示全面升级
- 全部 alert() 原生弹窗替换为底部 Toast 轻提示
- 教师端导航栏智能体异常通知改为底部 Toast

#### 课堂投屏二维码修复
- 课堂看板的投屏发码增加 ClassNode Logo 和下载按钮

#### 学生端 UI 优化
- 对话页顶部显示当前课堂互动码
- 修复多个浮动提示框撑大输入区域的问题

### 修复

- 修复 AI 请求无超时控制的 bug（新增 30 秒超时）
- 修复 Anonymizer 内存泄漏（设置 500 条上限自动重置）
- 修复课堂创建 N+1 查询问题（批量写入替代逐条 INSERT）
- 修复数据库备份为 0KB 的问题（DATABASE_URL 路径解析错误）
- 修复屏蔽词禁用后仍生效的问题（缓存 TTL 降至 3 秒）
- 修复活跃连接重连被拒绝的问题（自动断开旧连接）
- 修复学生输入框含英文/中文时不清除的 bug
- 修复打包脚本包含旧 logo 文件的问题
- 修复 make-dist.sh 端口注释与实际不一致的问题
