# 更新日志

## [1.3.6] — 2026-06-07

### 新增

#### 桌面端状态面板窗口
- 新增桌面窗口，实时显示服务运行状态、访问地址、局域网 IP
- 教师端/学生端地址一键复制与打开浏览器
- 系统托盘左键显示面板，右键弹出菜单
- 托盘菜单新增「显示面板」「项目地址」选项

#### 备份与恢复合并
- 备份整合数据库与附件（chat + logos），一键生成 .classbak 文件
- 恢复时自动解压并还原数据库和全部附件
- 历史备份列表支持上传 .classbak 跨设备迁移
- 备份大小自动显示 MB / KB

#### 内容过滤增强
- 系统屏蔽词和自定义屏蔽词支持一键启用/禁用
- 禁用后的屏蔽词以半透明显示

#### 弹窗提示升级
- 全部 alert() 替换为底部 Toast 轻提示，3 秒自动关闭
