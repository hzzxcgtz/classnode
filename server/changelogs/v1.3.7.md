# 更新日志

## [1.3.7] — 2026-06-08

### 新增

#### 投屏对话选择
- 右侧对话详情新增轮次选择功能，教师可逐轮勾选有价值的对话
- 投屏时仅展示被选中的轮次，未勾选的自动跳过
- 支持全选/取消全选，勾选提问或回答即选中整轮对话

#### 投屏界面重设计
- 投屏页全面改版，以轮次卡片形式展示对话
- 每轮对话独立成卡，学生提问和 AI 回答分列左右
- 顶部简化为"课堂标题 · 学习展示"风格

### 修复

#### 学生对话页
- 用户消息上方补充学生姓名标签
- 精简标签样式，去掉头像圆圈，缩小字号

#### 启动脚本
- Windows 启动脚本修复 `prisma` 命令找不到问题
- 改用 `npx` 解析本地依赖，兼容 pnpm node_modules 结构

#### 教师面板
- 右侧对话详情中 AI/学生名称移出气泡，作为独立标签
- 投屏页字号调小，避免喧宾夺主
- 屏蔽消息自动隐藏，不呈现在教师面板

#### 投屏与轮次逻辑
- 修复 roundIndex 为 0 时轮次选择框不显示
- 兼容旧数据 roundIndex 为空时自动补全轮次编号
- 修复 AI markdown 加粗 `**` 被误判为屏蔽消息
- 轮次选择改为位置计算，不依赖服务端字段
- 投屏选择栏固定在滚动区外，不随消息滚动

### 优化

- dev.sh 脚本彩色输出、URL 预览、构建计时等体验提升
- dev.md 使用说明同步更新
