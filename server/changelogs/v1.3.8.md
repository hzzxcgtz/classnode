# 更新日志

## [1.3.8] — 2026-06-08

### 新增

#### 屏蔽词拦截记录管理
- 拦截管理页新增「拦截记录」标签，按课堂汇总展示
- 支持单条删除和课堂级一键清空

#### 原生桌面窗口体验
- 原生菜单栏：macOS 屏幕顶部菜单 / Windows 标题栏菜单（文件、服务、帮助）
- 控制面板窗口完全重写，毛玻璃卡片设计，支持深色模式
- 服务状态动画指示器（运行中/已停止/启动中）

#### API Key 加密存储
- AES-256-GCM 加密 API Key 后存入数据库
- 首次启动自动生成随机加密密钥

### 优化

#### 屏蔽词性能优化
- 使用预构建 AC 自动机替代每次发消息重建
- 高并发场景下 CPU 开销显著降低
