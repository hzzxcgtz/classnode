# 更新日志

## [1.3.9] — 2026-06-09

### 新增

#### 文心智能体平台接入
- 新增「文心智能体」平台支持（百度文心），接入非流式对话 API
- 支持 App ID + Secret Key 认证，可在教师端添加和管理文心智能体

#### 平台管理优化
- 平台选择器改为：Coze 低代码、Coze 编程、文心智能体三大平台
- 预留更多平台接入位，后续可快速扩展
- 各平台使用柔和色调区分（灰蓝 / 灰紫 / 豆沙红），视觉更舒适

### 优化

#### 智能体表单体验
- 文心智能体 App ID 和密钥添加帮助图标，配图引导配置
- 各平台输入框提示文字按实际情况优化
- 文心智能体标记为「非流式」，表单中提示暂不支持流式输出
- 「从 Coze 获取」按钮仅 Coze 低代码平台显示

#### 智能体列表
- 平台标签配色柔和化，降低视觉刺激
- 文心智能体卡片显示「非流式」角标

#### 头像上传
- 支持 `.webp` 格式图片上传
