# 更新日志

## [1.4.0] — 2026-06-14

### 新增

#### 头像系统
- 头像管理页：支持随机生成头像、一键导入全部、逐个导入
- 学生头像：模块化卡通生成，含肤色、发型、校服、五官、配饰等维度自由组合
- 头像区分教师创建和学生自定义，教师可管理删除学生上传的头像
- 头像使用统计：底部数字角标显示使用次数

#### 教师奖励头像更换
- 课堂看板学生卡片操作栏新增⭐奖励按钮，奖励学生一次头像更换权限
- 获奖学生端实时弹出通知提示，姓名旁显示⭐徽章
- 班级管理学生列表也支持奖励操作

#### 学生自助换头像
- 三种换头像方式：选择教师头像库 / 粘贴自定义 SVG 代码 / 上传 JPG/PNG/WebP 图片
- 消耗教师奖励的 token，长期有效

### 优化

- 课堂看板学生头像：离线时自动灰度处理
- 学生列表行操作：编辑/删除改为图标按钮，新增清除头像按钮
- 批量操作：选中学生后支持批量清除头像
