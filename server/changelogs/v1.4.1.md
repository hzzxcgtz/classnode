# 更新日志

## [1.4.1] — 2026-06-14

### 新增

#### 学生性别
- Student 模型新增 gender 字段（boy/girl）
- 粘贴名单支持行内解析性别（姓名 空格/逗号 性别）
- 自动分配头像优先使用学生已设性别，无性别时按姓名推断

### 优化

- 逐个添加支持连续添加，完成后点"收起添加"关闭
- "自动分配头像"按钮：全班均已分配头像时自动隐藏

### 修复

- 数据库 schema 自动同步：服务启动时检测并创建缺少的表/字段
- 新增文件日志：所有控制台输出写入 `CLASSNODE_DATA_DIR/logs/`，便于排查问题
- 托盘右键菜单移除仓库链接
