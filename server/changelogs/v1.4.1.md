# 更新日志

## [1.4.1] — 2026-06-14

### 新增

#### 学生性别
- Student 模型新增 gender 字段（boy/girl）
- 逐个添加、粘贴名单、编辑学生均支持性别选择
- 粘贴名单支持行内解析性别（姓名 空格/逗号 性别）
- 学生列表新增性别列，支持点击排序
- 自动分配头像优先使用学生已设性别，无性别时按姓名推断

### 优化

- 头像管理页随机生成弹窗：头像分男孩/女孩区域，增加生成中旋转动画
- 底部三个操作按钮统一样式、高度一致，颜色简化
- 清空按钮移至女孩和学生自定义之间居中显示
- 逐个添加支持连续添加，完成后点"收起添加"关闭
- 添加学生姓名框缩短，性别改用 radio button 选择
- 编辑学生性别改用 radio button
- "自动分配头像"按钮：全班均已分配头像时自动隐藏
- 班级管理统计数字字号增大（0.813→1.125rem）

### 修复

- 数据库 schema 自动同步：服务启动时检测并创建缺少的表/字段
- 新增文件日志：所有控制台输出写入 `CLASSNODE_DATA_DIR/logs/`，便于排查问题
- 托盘右键菜单移除仓库链接
