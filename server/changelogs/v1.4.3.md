# 更新日志

## [1.4.3] — 2026-06-15

### 新增

#### 网卡选择持久化与全局同步
- 控制面板切换网卡后自动保存到数据库，全局生效
- 教师投屏页 URL 通过 Socket 实时同步网卡切换
- 服务端 `/api/server-info` 返回 `selectedIp` 和 `studentUrl`
- Tauri `cmd_open_url` 优先使用已保存的 IP 打开浏览器

#### 构建脚本重构
- `r` 系列命令：`./dev.sh r`（ARM64）、`r intel`（Intel）、`r both`（双架构）、`r all`（双架构+源码包）
- `release` 命令改为 Windows CI 构建（默认 x64），`release:full` 为全平台流程
- 移除 `release:intel` / `release:all` 旧别名
- 修复 `build:arm64` / `build:intel` 脚本名错误 → `build:mac:arm64` / `build:mac:intel`
- `build:mac` 构建时自动复制前端产物到 Tauri 资源目录

#### 下载测速
- `./dev.sh speedtest` — GitHub 下载速度测试，自动选取 Release 文件或源码 zip
- `--loop` 模式循环测速，方便切换代理链路对比速度

### 优化

- 教师首页和课堂管理页的投屏地址优先使用 API 返回的 `studentUrl`
- 投屏 URL 显示时去掉 `/classroom` 路径，扫码时保留
- 课堂看板去掉重复的二维码小按钮
- 网卡选择从教师网页移除，仅在 Tauri 控制面板中操作

### 修复

#### 备份恢复
- 备份时包含 `.encryption.key` 加密密钥文件，恢复后 API Key 可正常解密
- 加密密钥改为热重载机制，恢复后无需重启服务端
- 解决备份恢复后智能体测试请求失败的问题

#### 其他
- `./dev.sh r` 无参数时不再报 `unbound variable`
- 速度测试平均速度显示为 0 的问题
- 学生端消息框最大宽度从 78% 调整为 95%
