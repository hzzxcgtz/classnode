# 更新日志

## [1.4.7] — 2026-06-17

### 新增

* 服务端生成 DOCX：对话记录报表改为服务端生成，浏览器直接下载，大幅降低大数据量时的内存压力
* 按学生筛选导出：导出前可选择特定学生，仅导出选中学生的对话或报表数据
* 课堂恢复功能：已结束课堂支持恢复为活跃状态，自动重新生成互动码

### 改进

* 导出文件名优化：包含课堂名称和导出时间戳，更易识别
* 课堂看板页面新增班级名称显示

### 修复

* 移除 AI 回复中重复显示用户图片的问题
* 修复 Windows ARM64 安装后服务无法启动的问题 — 将 sharp 改为动态导入，避免 x64 原生模块在 ARM64 上加载崩溃；调整日志模块导入顺序确保任何启动错误可被记录
