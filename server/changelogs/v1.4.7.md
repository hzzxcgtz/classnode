# 更新日志

## [1.4.7] — 2026-06-17

### 新增

* 服务端生成 DOCX：对话记录和学情报表改为服务端生成，浏览器直接下载，大幅降低大数据量时的内存压力
* CSV 导出：对话记录和学情报表均支持导出为 CSV 格式，方便 Excel 打开处理
* 按学生筛选导出：导出前可选择特定学生，仅导出选中学生的对话或报表数据
* Socket 导出进度推送：服务端生成文档时实时推送进度百分比，前端展示进度条

### 改进

* 导出 DOCX 排版优化：页脚页码、移除互动码和智能体数量行、移除无效的消息构成占比
* 轮次分割统一从学生提问开始，到 AI 回答结束

### 修复

* CSV 单元格换行问题：所有单元格内容改为单行，避免 Excel 中部分换行部分不换行
* 对话图片附件嵌入：消息中的图片附件现在会嵌入 DOCX 文档，而非仅显示文件名
* 移除 v1.5.0 重复 changelog

### 技术变更

* 新增 `server/src/services/export-service.ts` 导出服务
* 新增服务端 DOCX/CSV 生成端点（4 个 POST 端点）
* 导出数据查询支持 `studentIds` 筛选参数
* 前端导出 UI 重构：学生选择、格式切换、进度条
