# 更新日志

## v1.5.0 (2026-06-21)

### 新功能
- **追问建议（follow-up）**：AI 回答后自动生成推荐追问问题，学生点击即可深入探讨
  - 目前：Coze 低代码、清言智能体支持该功能
- **教师看板新增控制开关**：教室看板工具栏新增四个控制按钮，实时控制学生端相关互动行为
  - 允许学生提问
  - 允许中断回答
  - 允许学生导出
  - 允许追问建议


### 修复
- **Coze 低代码接口实现重写，现已完美支持各功能**

### 改进
- **思考指示器动画**：波浪颜色根据 AI 回复内容动态变化，视觉反馈更丰富
- **对话页标记跳转**：支持点击标记滚动定位到对应消息
- **输入框升级**：从单行 input 改为 Textarea，支持多行输入和换行
- **屏蔽词语义优化**：改进屏蔽词匹配算法，减少误拦截
