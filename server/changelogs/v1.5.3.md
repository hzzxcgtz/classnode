# 更新日志

## [1.5.3] — 2026-06-28

### 新功能

**Linux PM2 一键部署**

从此版本起，ClassNode 正式支持 Linux 服务器部署，提供完整的安装/升级脚本和 PM2 进程管理方案。

- 新增 `install.sh` 安装/升级脚本：全中文引导，自动检测环境、安装依赖、构建项目、初始化数据库、配置 PM2
- 新增 `ecosystem.config.js` PM2 进程管理配置：支持开机自启、自动重启、日志轮转
- 版本升级时自动备份和恢复数据库、上传文件、加密密钥等用户数据，确保数据零丢失

**源码分发包改进**

- 源码包新增 `.tar.gz` 格式（Linux 原生支持），与 `.zip` 同时输出
- 源码包保留 `pnpm-workspace.yaml`，解压后可直接 `pnpm install` 构建，无需额外配置

### 文档

- README 新增 🐧 Linux 部署章节，包含完整的安装、升级、管理命令说明
