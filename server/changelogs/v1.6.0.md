# 更新日志

## v1.6.0 (2026-07-12)

### 安全增强
- **新增教师认证中间件**：敏感 API 路由强制鉴权，基于服务端会话管理
- **新增学生会话临时令牌**：JWT 风格令牌（2小时过期），页面刷新自动静默续领
- **密码存储升级**：SHA256 → scrypt 哈希，新增登录频次限制和暴力破解防护，密码最低长度 8 位
- **上传安全检测**：文件魔数验证、SVG XSS 清理、ZIP 路径穿越防护
- **代理密钥保护**：更新时不覆盖已加密的 API 密钥
- **CORS 与安全响应头强化**：credentials:true、X-Content-Type-Options、Referrer-Policy
- **Logo 上传安全加固**：文件名使用随机 UUID
- **数据库升级安全**：桌面端升级前自动创建备份快照，失败返回具体错误

### 代理配置重构
- AI 智能体配置页拆分为 12 个独立组件和自定义 Hook
- 新增 agent-card、credentials-fields、platform-selector、logo-field、help-button 等组件
- 新增 use-agent-controller、use-agent-form-actions 等 Hook
- 抽取平台配置常量，新增类型定义

### 教室与学生端升级
- 学生端 API 调用自动携带 Bearer 令牌，Socket.IO 启用 withCredentials
- Typed Socket Events（ServerToClientEvents / ClientToServerEvents）
- 创建教室参数校验强化，分组模式支持 Prisma 事务幂等创建
- 新增 POST /code/:code/student-session 无感登录端点

### 导出与设置重构
- 数据库恢复前自动创建安全快照，支持还原回滚
- 新版 .classbak 解压增加安全限制
- 设置路由重构：新增 init-status 端点、登录限频、会话管理

### 前端优化
- About 页面全新视觉设计（SectionCard 卡片组件、渐变色）
- Layout 侧边栏改用服务端验证
- 屏蔽词管理、班级管理、教室板、聊天界面等多处交互优化
- 教师主页快速入门引导优化

### 测试
- 新增 server/src/tests/ 测试目录，覆盖安全性、上传检测、教室状态等

### 文档
- 新增 AGENTS.md 项目说明文档
