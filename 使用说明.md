# ClassNode · AI 互动课堂 — 使用说明

> 版本 0.1.0 · 支持 macOS / Windows

---

## 一、准备工作

安装 **Node.js 18 或更高版本**（LTS 版即可）：
- 下载地址：https://nodejs.org
- 安装时全部默认选项，一路"下一步"

---

## 二、启动应用

找到 ClassNode 文件夹，**双击启动脚本**：

| 你的系统 | 双击 / 运行这个文件 |
|---|---|
| macOS | `start-classnode-mac.command` |
| Windows | `start-classnode-win.bat` |
| Linux | 终端执行 `bash start-classnode-linux.sh` |

首次启动会自动完成依赖安装和构建（需联网，约 1-5 分钟），之后启动会快很多。

启动成功后，浏览器会自动打开教师端页面。

---

## 三、开始使用

| 入口 | 地址 |
|---|---|
| 教师端 | `http://localhost:3000/teacher` |
| 学生端 | `http://localhost:3000/classroom` |

首次进入教师端，按页面提示设置管理密码，然后配置 AI 智能体、创建班级和课堂即可。

> 如果需要在**其他设备**上访问（如学生手机），请将 `localhost` 替换为教师电脑的局域网 IP 地址，例如 `http://192.168.x.x:3000/classroom`。

**关闭应用：** 直接关掉服务窗口（Windows）或按 `Ctrl+C`（macOS）即可。

---

## 四、常见问题

**Q：双击脚本后没反应？**
Node.js 没有安装或安装不正确，请先安装 Node.js。

**Q：提示端口被占用？**
关闭其他正在运行的程序，或者重启电脑后再试。

**Q：如何重置所有数据？**
进入教师端「历史数据」页面，点击「初始化清零」。或者直接删除 `server/prisma/dev.db` 文件，重启后会自动重建。
